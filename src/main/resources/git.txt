将项目通过git来进行管理
1.自己创建好基本项目结构
2.将我们的项目变成一个git仓库 git init
3.因为代码仓库只需要存放我们的代码
  像class文件 编译器自身生成的文件都不需要放入仓库中
  解决方案就是在忽略文件中进行设置 .gitignore
4.在设置好忽略文件后 就可以将代码提交到我们的本地仓库
  这时候对于我们的文件需要先通过 git add 要添加的文件的路径
  来对放入本地仓库的代码进行添加操作
  接下来通过git commit - m "提交信息" 来真正的将我们的项目代码提交到本地仓库
  这时候才是真正的将代码保存在本地仓库了
  本地仓库和远程git仓库进行连接并将代码推送到远程服务器上
  这样才能跟别人进行共同开发
  git remote add origin 远程服务器上面的仓库的地址
  连接好后 就可以把项目推送到本地服务器
  git push  -u origin master
